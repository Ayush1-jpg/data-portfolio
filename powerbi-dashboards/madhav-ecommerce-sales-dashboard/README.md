# Madhav Ecommerce Sales Dashboard

This Power BI dashboard analyzes sales performance, profit trends, and customer purchasing patterns for an ecommerce business.

## Tools Used

- Power BI
- Data Visualization
- Data Analysis

## Dashboard Overview

This dashboard provides insights into:

- Total sales and profit
- Sales by state
- Sales by category and sub-category
- Monthly profit trends
- Payment mode analysis
- Top performing products
